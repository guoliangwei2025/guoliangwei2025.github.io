Python中的哈希

哈希表基础

哈希表（Hash Table） 是一种 基于键值映射的数据结构，核心特点：

  • 键（key） → 哈希函数 → 数组索引 → 值（value）
  • 查找、插入、删除平均时间复杂度 O(1)
  • Python 内部使用 开放寻址 + 链表/红黑树 来处理冲突

dict（字典）

Python中内置的哈希表类型

key：不可变的类型（int，str，tuple）

value：任意类型

defaultdict（带默认值的字典）

来自来自 collections 模块，工厂函数指定默认值类型

常用类型：

  • int → 默认 0，用于计数
  • list → 默认 []，用于分组
  • set → 默认 set()，用于去重

Counter（计数器）

  • collections.Counter 是 dict + int 封装的计数工具
  • 功能：统计元素频率，直接返回哈希表

set（集合）

哈希实现的无序集合，元素唯一、查找快速，常用于去重、快速判断元素是否存在

哈希使用场景

场景 常用类型 说明
元素去重 set 判断元素是否存在
计数/频率统计 dict / Counter / defaultdict(int) 统计出现次数
分组/归类 defaultdict(list) 通过 key 分组元素
查找/索引映射 dict key → value 快速查找
